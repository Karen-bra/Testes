# SCA- Sistema de Controle de Alunos 

`Análise e Desenvolvimento de Sistemas - PUC Minas`

`Desenvolvimento de aplicação web front-end`

`1º semestre/2025`

O objetivo principal desse projeto e testar as minhas habilidades que desenvolvi durante o curso, essa aplicação foi pensanda em cada matéria que estudei no decorrer desse semestre, estou criando uma aplicação para professores na ajuda do controle de notas de semestre de cada aluno.
